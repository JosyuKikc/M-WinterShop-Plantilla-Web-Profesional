## ✨ Características

### 🎨 Diseño & UI
- 🌑 **Tema dark premium** con acentos en violeta/gradientes
- ✨ **Animaciones suaves** en scroll (Intersection Observer API)
- 📐 **100% Responsive** — Desktop, Tablet y Móvil
- 🔤 **Tipografías modernas** — Inter + Space Grotesk (Google Fonts)
- 🎭 **Iconografía completa** — Font Awesome 6.5

### 🛒 E-commerce
- 🛍️ **8 productos** con badges (Nuevo, Hot, -40%)
- ❤️ **Wishlist toggle** en cada producto
- 🛒 **Carrito sidebar** con contador animado
- 🏷️ **Sistema de precios** con descuento y precio original
- 📂 **Filtro por categorías** con tabs interactivos

### 🧩 Secciones
| Sección | Descripción |
|---------|-------------|
| 🎬 Loader | Pantalla de carga animada con logo |
| 📢 Promo Bar | Barra superior con countdown en vivo |
| 🧭 Navbar | Fija, transparente → blur al scroll, menú hamburguesa |
| 🔍 Buscador | Modal con input y cierre con ESC |
| 🏠 Hero | Orbs animados, stats, CTAs, tarjetas flotantes |
| 📜 Marquee | Banda infinita con beneficios |
| ⭐ Features | 4 cards (envío, pago seguro, cambios, soporte) |
| 👕 Colecciones | 3 categorías con overlay y badges |
| 🛍️ Productos | Grid de 8 productos completos |
| 🔥 Banner Promo | Con temporizador regresivo de 3 días |
| ⭐ Reseñas | 3 reviews + resumen con barras de rating |
| ❓ FAQ | 6 preguntas con acordeón animado |
| 📧 Newsletter | Formulario de suscripción al email |
| 📱 Footer | Links, redes sociales, métodos de pago |
| 💬 WhatsApp | Botón flotante con pulse animation |
| ⬆️ Back to Top | Aparece al hacer scroll |

### ⚡ Funcionalidades JS
- ⏰ Countdown timer en tiempo real (promo bar + banner)
- 🛒 Carrito interactivo con sidebar
- 🔍 Modal de búsqueda (abre/cierra con ESC)
- 📱 Menú hamburguesa responsive
- ❤️ Toggle de favoritos por producto
- 🎯 Scroll reveal con IntersectionObserver
- 🔗 Smooth scroll en navegación
- 📌 Navbar con efecto glassmorphism al scroll
